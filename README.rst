# Sharing secret
Implementation of the [Shamir's secret sharing]{https://en.wikipedia.org/wiki/Shamir%27s_secret_sharing}


## Installation

You can install the package via Git,

```bash
git clone https://github.com/adriangolmayo100/sharing_secret
```

## Usage

```bash
python3 run.py
```
